# TON 문서

TON 블록체인 개발 공식 문서에 오신 것을 환영합니다!

## DApp 개발

분산 응용 프로그램 (DApp)은 하나의 컴퓨터가 아닌 컴퓨터 피어 투 피어 네트워크에서 실행되는 응용 프로그램입니다 (TON 블록체인). 이것들은 전통적인 웹 애플리케이션과 유사하지만 블록체인 네트워크 위에 구축됩니다. 즉, DApp은 분산되어 하나의 엔터티가 제어하지 않는다는 것을 의미합니다.
